Adds visual category description editor (TinyMCE).


== Description ==

This plugin enables the WordPress visual editor TinyMCE on the Edit Category and Edit Tag admin pages. 


== Installation ==

1. Download.
2. Unzip.
3. Upload the "cat-description-editor" folder to the plugins directory and activate the plugin.
4. Edit any category or tag to see the changes.
